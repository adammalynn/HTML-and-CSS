Summary: Lattitude vs. X


In this project, we analysed weather changes on January 5th 2017 over 500 cities. Using data from OpenWeatherMap API, we collected and cleaned the dataset before creating Visualizations using Matplotlib. We observed changes in the weather accross different Lattitude using specific weather components like Temperature, Wind, Humidity and Cloudiness.
